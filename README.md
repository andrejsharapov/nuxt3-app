## Create a new starter project

```bash
npx nuxi init nuxt3-app
cd nuxt3-app
yarn install
```

## Pre-processors

```bash
yarn add --dev pug pug-plain-loader
yarn add --dev sass sass-loader@10
```

## Nuxt modules

```bash
yarn add --dev @nuxtjs/color-mode
```
